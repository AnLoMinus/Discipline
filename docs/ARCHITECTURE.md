# אדריכלות המערכת

## מבט-על

המאגר `Discipline` מרכז תיעוד, נהלים ומדריכים בנושא משמעת עצמית. לצד המסמכים קיימת מעטפת אתר סטטי לקריאה נוחה.

```
Discipline/
├─ docs/              # כל מסמכי ה-Markdown
├─ index.html         # דף בית סטטי
├─ guide-discipline.html
├─ styles.css
└─ script.js
```

## עקרונות תכנון

- **דגש על תוכן**: כל הערך של המאגר נמצא במסמכים כתובים ובחוויית קריאה נוחה.
- **Zero Install**: אין תלות בקוד או בהתקנות; כל הקבצים ניתנים לצפייה בדפדפן או בעורך Markdown.
- **תיעוד מובנה**: שימוש ב־`docs/INDEX.md` כעמוד שער, וב־ADR's לתיעוד החלטות.

## מפת דרך (טיוטה)

1. הרחבת מדריכים ייעודיים ב־`docs/guides/`.
2. הפקת גרסאות PDF/Poster מתוך המסמכים המרכזיים.
3. שיפור חוויית האתר הסטטי (אנימציה, ניווט משופר).

עדכונים נוספים יתועדו ב־[`docs/adr`](adr/README.md) וב־[`ROADMAP.md`](../ROADMAP.md).

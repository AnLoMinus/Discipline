# התחלה מהירה

## דרישות מוקדמות
- Git 2.40+
- Python 3.11+ (או Node.js 20+, בהתאם לרכיבי הקוד שתבחר להפעיל)
- Make או כלי ניהול משימות דומה (אופציונלי)

## התקנה
```bash
git clone https://github.com/AnLoMinus/Discipline.git
cd Discipline
```

## סביבה
- צור קובץ `.env` מקומי אם נדרש.
- הרץ את הסקריפטים מתוך `scripts/`.
- בדוק את הממשק החי: [https://anlominus.github.io/Discipline/](https://anlominus.github.io/Discipline/).

## בדיקת התקנה
- הרץ בדיקות: `make test` או `scripts/run-tests.sh`.
- הפעל לינט: `make lint` או `scripts/run-lint.sh`.

## המשך
- קרא את [`CONTRIBUTING.md`](../CONTRIBUTING.md).
- עבור אל [מדריך האדריכלות](ARCHITECTURE.md) להבנת המודולים.

# התחלה מהירה

המאגר מכיל תיעוד ומדריכים בלבד – אין צורך בהתקנות או בהרצת קוד.

## איך לצרוך את התוכן?
1. בקרו באתר החי: [https://anlominus.github.io/Discipline/](https://anlominus.github.io/Discipline/).
2. אם תרצו גרסה מקומית, שיבטו את המאגר ופתחו את `index.html` או כל קובץ Markdown בעורך טקסט.
3. השתמשו ב־[`docs/INDEX.md`](INDEX.md) כעמוד שער לכל המסמכים.

## המשך
- קראו את [`CONTRIBUTING.md`](../CONTRIBUTING.md) כדי לדעת איך לעדכן את המסמכים.
- עיינו ב־[`ROADMAP.md`](../ROADMAP.md) להבנת הכיוון האסטרטגי.
